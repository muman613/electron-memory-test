Instructions

This application was developed to demonstrate a limitation in the Electron/NodeJS environment.

The Emscripten transpiler implements the application heap as an ArrayBuffer object. This application will show that any attempt to allocate a heap of greater than or equal to (>=) 2Gb results in a 'RangeError: Array buffer allocation failed' exception.

In the Memory Parameters section below enter a value for Memory Size and select the memory prefix (Gb/Mb/Kb/b) from the select control. Hit the 'Allocate' button to attempt to allocate the memory. If it suceeds a message is displayed in the status section, otherwise an alert is displayed with the Exception message.
